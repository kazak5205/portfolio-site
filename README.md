# Portfolio Website

Portfolio website for international freelancing. Built with modern web technologies.

## Structure

- **Home:** Hero section, what I do, CTA (Hire/Contact)
- **Cases:** 3 project cards with images, descriptions, live demos, results
- **About:** Photo + 3-5 competency points
- **Contact:** Email, Telegram, Calendar link (Calendly/Google)

## Tech Stack

- Next.js / React (for GitHub Pages: static HTML/CSS/JS)
- Or Vercel for easy deployment
- Tailwind CSS for styling
- Responsive design

## Deployment

### Option 1: GitHub Pages
1. Build static site
2. Push to `gh-pages` branch
3. Enable GitHub Pages in repo settings

### Option 2: Vercel
1. Connect GitHub repo
2. Deploy automatically
